# Instagram Clone

A responsive Instagram Clone built using HTML and CSS.

## Features

- Responsive Layout
- Navigation Bar
- Stories Section
- Feed Design
- Suggested Profiles
- Modern Instagram-inspired UI

## Technologies Used

- HTML5
- CSS3
